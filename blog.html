Becoming a front end developer

June 14, 2015

I would like to talk about becomming a front end developper.

Differently from other technologies I have previously learned, learning front end seems to be a very bumpy road.

Whenever I have learned how to write applications with other technologies I had one or maybe a few different ways of doing one thing. I could read several books and they would pretty much go the same direction. Now this is different. Every book bumps me in a different but somewhat similar direction. That can be quite confusing and somewhat frustrating. More on that later.
